Add HTML, CSS, JS and PHP snippets to your content via shortcodes.

== Description ==
A simple WP plugin which allows you to add HTML, JS, CSS or PHP snippets to your page. You can find the development over at GitHub: https://github.com/nilshellerhoff/wordpress-snippets


== Installation ==
Install and activate. Some code examples will show up on the \"Snippets\" admin menu.
